# Hash Project

Небольшой проект на Python по хеш-таблицам и хеш-функциям.

## Что есть в проекте

- класс `HashTable`
- операции `insert`, `search`, `delete`
- метод `resize` для увеличения таблицы
- простая хеш-функция для строк
- словарь, где ключи — строки, а значения — их хеши

## Файлы

- `hash_table.py` — класс хеш-таблицы
- `string_hash.py` — функция хеширования строки
- `string_hash_dict.py` — словарь строк и их хешей
- `main.py` — запуск примеров
- `results.txt` — результаты работы программы

## Как запустить

```bash
python main.py
```
